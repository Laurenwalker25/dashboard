NCAA Men's Basketball Dashboard

About this project

This site serves as a hub for information regarding 2024-25 NCAA Men's Basketball. It current reports the following statistics from each week's AP Top 25 teams for weeks 14 and on:

  • Team: Team name
  • Record: Team record (W-L)
  • Wins (%): Percentage of games won
  • Conference: Team conference
  • Scoring Offense Rank: Rank in scoring offense metric
  • Scoring Offense (PPG): The average number of points a team scores in a game
  • Scoring Defense Rank: Rank in scoring defense metric
  • Scoring Defense (PPG): The average number of points a team's defense allows in a game
